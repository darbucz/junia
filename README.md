# LabrApp 🐶

Testowa aplikacja webowa dla miłośników labradorów.

## 🔧 Jak uruchomić

1. Zainstaluj zależności:

```
npm install
```

2. Uruchom lokalny serwer:

```
npm run dev
```

3. Otwórz przeglądarkę i wejdź na `http://localhost:5173`

## 💡 Funkcje

- Profil labradora
- Galeria zdjęć
- Mini poradnik

Made with 💙 using React + Tailwind + Vite.
